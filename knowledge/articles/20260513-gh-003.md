## 摘要

trycua/cua 是专为 Computer-Use Agent 场景设计的开源基础设施平台，将沙箱运行环境、开发 SDK 和评测基准整合在单一项目中。这类基础设施的成熟标志着 AI Agent 从文本对话向真实桌面操控的能力延伸进入可规模化研究阶段。对于需要构建 RPA 类 Agent 或探索 GUI 自动化的团队，cua 提供了一套开箱即用的实验平台。其基准测试体系也为评估不同模型在桌面操控任务上的表现提供了统一标准。

## 核心亮点

- 提供完整的 Computer-Use Agent 基础设施，包含沙箱、SDK 和评测基准三位一体
- 支持训练和评估能够控制完整桌面环境的 AI Agent
- 开源实现降低了 Computer-Use 能力研究门槛
- 基准测试体系可量化衡量 Agent 桌面操控能力进展
